# リモートの状態を確認する。  
```
$gir remote
```
対応するURLを表示 
```
$git remote -v
```
# リモートリポジトリを追加する。  
リモートリポジトリを複数登録できる。  
<リモート名>は任意の名称を指定できる。  
```
$git remote add <リモート名> <リモートURL>
```
複数のリモートリポジトリを登録すると、`$git remote`を実行すると複数のリモート名が表示される。
# リモートから情報を取得する(フェッチ)  
リモートリポジトリの情報はワークツリーには入ってこないが、ローカルリポジトリに保存される。
```
$git fetch <リモート名>
$git fetch origin
```
ローカルリポジトリの情報をワークツリーに落としてきたい時
```
$git merge
```
